PDMS COMPOSITE MATERIAL AUTOMATIC TYPESETTING

GENERAL INTRODUCTION

总体介绍


PDMS design software is a factory, the main function is the 3 d pipeline design. Can automatically generate the equipment layout, piping layout, ISO figure, composite material table, etc. Make use of the automatic generation of ISO figure is more perfect, generally have generated CAD equipment layout, piping layout needs to modify a moderate amount of composite materials, table is very chaotic, need manual layout (period error prone), due to various reasons after typesetting often need to modify the model to the material, recomposition, so you need a lot of time. In order to improve the design quality and efficiency, affiliated with EXCEL VBA language developed "composite material (Ver1.0) automatic typesetting system".
